##Pizza App

A modern pizza ordering application built with React.
Users can browse pizzas, add items to cart, and place orders with a smooth user experience.

##Live Demo

https://pizzadattayf.netlify.app/

##Overview

This project focuses on building a real-world ordering flow with global state management, routing, and asynchronous data handling.

##Features
Browse pizza menu
Add / remove items from cart
Update quantity
Order summary
Place order
Responsive UI

##Tech Stack
React
React Router
State management (Context API / Redux - Thunk)
CSS / Styled Components

##Scripts

npm run dev
npm run build
npm run preview
